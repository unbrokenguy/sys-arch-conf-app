## Use python >= 3.7

## Install poetry
```shell
pip install poetry
```

## Django Conf App

#### Install the project dependencies
```shell
poetry install 
```

#### Spawn a shell within the virtual environment
```shell
poetry shell
```

#### Start server
```shell
python manage.py runserver 8001
```
